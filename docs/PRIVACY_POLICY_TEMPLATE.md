## Data Storage and Security

### Where Your Data is Stored
- **Cloud Storage**: Your data is stored on Firebase (Google Cloud Platform) servers
- **Local Storage**: Some data may be cached locally on your device for offline access

### Security Measures
We implement appropriate technical and organizational security measures:

- **Encryption**: Passwords are hashed using SHA-256 with salt
- **Secure Storage**: Sensitive data uses encrypted storage (expo-secure-store)
- **Access Controls**: Firebase security rules restrict data access
- **Input Validation**: All user inputs are sanitized to prevent injection attacks

However, no method of transmission over the internet or electronic storage is 100% secure. We cannot guarantee absolute security.
